pip update-all

2019-03-15
pip python one-liner

Problem to solve

Update all packages which are managed by pip.

Fix

pip list --outdated --format=freeze | grep -v '^\-e' | cut -d = -f 1  | xargs -n1 pip install -U

Alternatively

pip freeze|grep '=='|awk -F= '{print $1}' | xargs pip install -U
